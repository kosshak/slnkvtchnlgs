# Помехоустойчивая технология обработки сигналов для управления БПЛА

Программно-аппаратное решение для устойчивого управления БПЛА в условиях РЭБ на основе голографического кодирования, DFT-s-OFDM и формирующей фильтрации.

## Технологии
- Голографическое кодирование
- DFT-s-OFDM
- Формирующая фильтрация
